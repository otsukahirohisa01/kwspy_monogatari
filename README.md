## Requirement
- Windows OS
- Python 3.7
- venv (bundled in Python 3.7)

## How to run
- Edit "PLAYER" variable in player.py based on player name you created
- Open cmd.exe and execute below command

```
$ .\env\Scripts\activate.bat
$ python player.py
```
